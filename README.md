# Be Better - 个人学习计划管理平台

Be Better 是一个帮助用户管理和追踪学习计划的 Web 应用程序。它提供了直观的界面和丰富的交互体验，帮助用户更好地规划和执行学习计划。

## 功能特点

- 📚 学习计划管理
  - 创建自定义学习计划
  - 按天划分学习任务
  - 可视化的学习进度追踪
  
- 🎯 任务完成度追踪
  - 任务打卡功能
  - 进度条显示
  - 完成动画和庆祝效果
  
- 👤 用户系统
  - 用户登录/注册
  - 个人计划管理
  - 头像和个人信息设置

## 技术栈

- 前端框架：Vue 3
- 开发语言：TypeScript
- 构建工具：Vite
- UI 组件库：Naive UI
- 路由管理：Vue Router
- 动画效果：Canvas Confetti
- 图标库：Ionicons 5
